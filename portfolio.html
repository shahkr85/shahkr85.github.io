Custom Composite DLP Printing

Custom resin 3D printer developed to create composite parts out of resin and fiberglass mats. Full resin DLP printing functionality has been achieved, and we are now developing a belt-drive system to incorporate the composites.

ADS-B Flight Tracker

Using an RTL-SDR & an open source software "Dump1090", flights within a local area were tracked. The main goal of this project is to manually decode various ADS-B signals received by the RTL-SDR and analyze the algorithms used by Dump1090 to decode.
